## Описание проекта
### Идея
На сегодняшний день многие люди начинают или уже занимаются различными видами спорта для поддержания своего физического состояния. Но в силу разных факторов люди не могут получить достойную информацию о тренерах и программах тренировок, а квалифицированные тренеры не могут найти себе клиентов. Данное решение поможет избавиться от этих двух проблем.

TrainHard — платформа, которая объединяет клиентов, ищущих персональные/групповые тренировки или готовые программы, и тренеров, предлагающих свои услуги. Пользователи могут искать тренеров по различным критериям (например, направление, опыт, цена, локация), просматривать программы тренировок, записываться на занятия и покупать программы онлайн.

### Целевая аудитория
- **Клиенты**: люди, желающие начать заниматься спортом, улучшить физическую форму, похудеть или подготовиться к соревнованиям.
- **Тренеры**: фитнес‑тренеры, инструкторы по йоге, пилатесу, единоборствам и другим дисциплинам, которые хотят расширить клиентскую базу.

### Какие задачи решаются
- Упрощает поиск квалифицированных тренеров с проверенной информацией об опыте и квалификации.
- Позволяет сравнивать тренеров и программы по цене, отзывам, специализации.
- Даёт тренерам инструмент для продвижения услуг без затрат на маркетинг.
- Повышает прозрачность рынка фитнес‑услуг за счёт рейтингов и отзывов.

### Ключевые экраны и функциональность
#### Для клиента (роль: `client`)

| Экран | Что можно делать (наполнение) |
|-|-|
| Главная | Поиск тренеров/программ; фильтр по категориям, локациям, цене; блок "Популярные тренеры" с карточками (фото, имя, рейтинг, направление); блок «Хиты продаж» (готовые программы с описанием) |
| Каталог тренеров | Список тренеров с пагинацией; расширенные фильтры: опыт (лет), сертификаты, онлайн/оффлайн занятия; сортировка: по рейтингу, цене, новизне |
| Карточка тренера | Фото, имя, стаж, сертификаты (сканы); видео‑презентация; список программ тренера; отзывы клиентов с рейтингами; кнопка "Записаться на тренировку" или "Связаться" |
| Страница программы тренировки | Название, описание, длительность, уровень сложности; план тренировок по неделям/дням; цена и кнопка "Купить"; отзывы о программе |
| Личный кабинет клиента | История заказов (купленные программы); график запланированных тренировок; форма отзыва о тренере/программе; настройки профиля |

#### Для тренера (роль: `coach`)

| Экран | Что можно делать (наполнение) |
|-|-|
| Личный кабинет тренера | Редактирование карточки тренера; статистика: доход, количество учеников, рейтинг; форма создания/редактирования программы; уведомления о сообщениях и купленных программ; общение с клиентами |
| Форма создания/редактирования программы | Поля: название, описание, цена, длительность, уровень сложности; загрузка PDF-файлов или видео материалов; распределение уроков по дням/неделям |

#### Для администратора (роль: `admin`)

| Экран | Что можно делать (наполнение) |
|-|-|
| **Admin-панель** | управление клиентами/тренерами/программами тренировок; модерация данных клиентов/тренеров/программ и отзывов;  |

## User Stories
```
Как клиент, я хочу искать тренеров по направлению (йога, кроссфит и т. д.), чтобы найти специалиста под свои цели.

Как клиент, я хочу видеть рейтинги и отзывы о тренерах, чтобы принять осознанное решение.

Как клиент, я хочу покупать готовые программы тренировок онлайн, чтобы заниматься самостоятельно.

Как тренер, я хочу создать профиль с указанием опыта, сертификатов и видео‑демонстрацией занятий, чтобы привлечь клиентов.

Как тренер, я хочу публиковать готовые программы тренировок с описанием и ценой, чтобы монетизировать свой опыт.

Как тренер, я хочу получать уведомления о новых заявках и сообщениях от клиентов, чтобы оперативно реагировать.

Как администратор, я хочу модерировать профили тренеров и отзывы, чтобы поддерживать качество платформы.

```

## Сущности системы
| **Сущность** | **Ключевые поля** |
|-|-|
| users | id, email, password_hash, role (client/trainer/admin), created_at |
| profiles | user_id, name, bio, experience_years, certificates, photo_url, rating |
| programs | id, trainer_id, title, description, price, duration_weeks, difficulty_level, category |
| sessions | id, trainer_id, client_id, date_time, location, status |
| orders |  id, user_id, program_id, amount, status |
| reviews | id, author_id, target_id, rating, text, created_at |
